# Sounds

Place `.wav` audio files here. They are served at `/sounds/<filename>`.

| File | Event |
|---|---|
| `match-start.wav` | Auto period begins |
| `auto-end.wav` | Auto period ends (transition buffer starts) |
| `teleop-start.wav` | Teleop period begins |
| `match-end.wav` | Teleop ends and match completes normally |
| `match-abort.wav` | Match is aborted mid-run |
